# 大学生请假管理系统 python956

## 项目概述

大学生请假管理系统是基于Python+Django开发的，旨在简化和规范请假流程，提高管理效率。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

- 学生信息管理
- 辅导员信息管理
- 班级信息管理
- 请假申请
- 请假审批与管理

## 系统角色

- 学生：发起请假申请
- 辅导员：审批请假申请并进行管理

## 运行环境

- Python 3.8+
- Django 3.2+
- MySQL 5.7+
- Vue.js 2.6+

## 部署步骤

1. 安装Python环境和相关依赖
2. 安装MySQL数据库，并导入初始数据
3. 部署Django后端服务
4. 部署Vue.js前端服务

## 目录结构

```
project/
├── backend/               # 后端代码
│   ├── apps/              # 应用模块目录
│   ├── manage.py          # 管理脚本
│   └── ...
└── frontend/              # 前端代码
    ├── src/               # Vue源码目录
    ├── public/            # 公共文件目录
    └── ...
```

## 核心亮点

- 灵活的角色权限管理，满足不同用户需求
- 基于Django的后端，易于维护和扩展
- 使用Vue.js的前端，界面友好，响应迅速
- 全套文档支持，方便用户了解和部署系统
